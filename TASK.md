Your task is to build a simple IFC viewer running in a web browser.

IFC (Industry Foundation Classes) is the standard file format to describe a building. 
Find more information here: https://www.buildingsmart.org/standards/bsi-standards/industry-foundation-classes/

Use the xeokit SDK to build the viewer: https://github.com/xeokit/xeokit-sdk

Here's an example on how xeokit is used to display IFC files: https://github.com/xeokit/xeokit-sdk/blob/master/examples/buildings/web-ifc_vbo_Duplex.html
And a few more examples: https://xeokit.github.io/xeokit-sdk/examples/

Features of our viewer:
 * Simple SPA, use JavaScript but no frameworks
 * Everythin runs on the client-side, no API usage
 * User uploads .ifc file which is then rendered in the browser
 * Simple UI
 * Mobile friendly
